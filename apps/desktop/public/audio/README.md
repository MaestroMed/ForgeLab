# Audio Files for FORGE LAB

This directory contains audio files for the ambient audio system and sound effects.

## Required Files

### Ambient Tracks (`ambient/`)

Place your ambient audio files here. These should be:
- Format: MP3
- Duration: 3-5 minutes (will loop seamlessly)
- Style: Ambient, cinematic, non-intrusive

| File | Description |
|------|-------------|
| `westworld.mp3` | Cinematic ambient inspired by Westworld OST (Ramin Djawadi style) |
| `minimal.mp3` | Subtle drone, very low-key |
| `deep.mp3` | Deep bass ambient |

### Sound Effects (`sfx/`)

Short sound effects for notifications:

| File | Description |
|------|-------------|
| `notification.mp3` | General notification ping |
| `success.mp3` | Export/task completed successfully |
| `error.mp3` | Error occurred |
| `swoosh.mp3` | Transition/intro animation sound |

## Recommended Sources

- **Freesound.org** - Free sound effects (CC licensed)
- **Artlist.io** - Professional music library (subscription)
- **Epidemic Sound** - Royalty-free music (subscription)
- **YouTube Audio Library** - Free music for creators
